# WHACK-A-MOLE

**A simple whack-a-mole game made using React**

# Purpose

    This project is based on the code test requested by Gaming1.

# Prerequisites

-   `node.js`
-   `yarn`

## Usage

-   After cloning the repo

    ```
    cd whakeamole
    yarn build
    yarn start
    ```

-   Open http://localhost:3000 on your Browser
